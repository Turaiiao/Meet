# The Meet Programming Language

```
var her: string = '余春莉', me: string = '黄菁'

for var i = 1; i <= 10000; i += 1 {
    if i <= 1500 -> println '$me 爱 $her $i 遍 !'
    
    else ->
        println '$me 最爱 $her $i 遍 !!!'
        
    if i == 3000 {
        println '\n 哈哈，先来个 3000 遍，10000 遍以后再说 \n'
        
        break
    }
}
```
